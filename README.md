# Install project

### Install dependencies
```
npm install
```
# Run project

### Build for DEV environment
```
npm run build-dev
```
### Build for PROD environment
```
npm run build-prod
```
### Run server
```
npm run server
```
### Run watch
```
npm run watch
```
